# FindThemNow — AI-Powered Global Missing Persons Platform

FindThemNow is a non-profit, global platform designed to assist in finding missing persons through real-time AI biometric matching and seamless coordination with local law enforcement across 80+ countries.

## 🚀 Key Features

- **Biometric Matching**: AI-powered analysis using 12+ markers including facial recognition, iris patterns, fingerprints, and physical descriptors (scars, moles, tattoos).
- **Live Camera Scanner**: Real-time camera search against the global missing persons database.
- **Worldwide Emergency Services**: One-tap access to police, ambulance, and fire services for 60+ countries.
- **Embassy Finder**: Quick access to consular services for citizens abroad.
- **Active SOS**: Instantly share GPS location with local authorities and emergency contacts.
- **Live Safety Alerts**: Real-time disaster, conflict, and health advisories worldwide.

## 🛠️ Technical Improvements (v2.0)

- **Architecture**: Refactored to a modular, single-page application (SPA) architecture with non-blocking script loading (`defer`).
- **PWA Ready**: Enhanced Service Worker with improved caching strategies (Network-first for HTML, Cache-first for assets) and offline support.
- **Accessibility**: Full WCAG 2.1 compliance including skip-links, ARIA labels, focus traps for modals, and removal of zoom restrictions.
- **Security**: Implemented input sanitization to prevent XSS and added client-side form validation.
- **UX/UI**: Added skeleton loaders for smooth transitions, real Web Geolocation for SOS, and a persistent dark mode.

## 💻 Tech Stack

- **Frontend**: Vanilla JavaScript (ES6+), CSS3 (Custom Variables), HTML5.
- **PWA**: Web App Manifest, Service Workers.
- **APIs**: Web Geolocation API, Web Camera API.
